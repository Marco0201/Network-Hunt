Q & A

What tech's did you use to make this website?

Great question! This website was created using the following technologies: HTML, CSS, JavaScript, and Bootstrap for the frontend; Python and Django for the backend; and PostgreSQL as the database.

Why did you make this website?

Another great question! This site was created to manage the job applications I applied for and to track the networking I did during the job hunt. I decided to make this because I didn't really like tracking everything on excel or word. Don't get me wrong, Excel and Word are versatile and great tools, but I just don't like using them. So I made this.

Is there a way to make the notes section larger?

Yes! There is a button on the notes section's toolbar that allows you to go fullscreen. It's located under the file dropdown menu. I would also encourage you to look at the insert menu options, especially for developers; you can insert code into the notes section. Although you can't run the code, it enables you to format the inputted code.

Can I upload imgs and media files in the Notes section?

Fantastic question! Simple answer: No. You can include images and media files in the notes section, but they must be hosted elsewhere. You can insert them using a link, but local uploads aren't possible. I chose not to include this option to reduce strain on the backend.

Why did you not include a regular registration and login form?

Wow, you have so many great questions! The idea behind not including a regular registration and login form is to simplify things on the backend. I aim to let the social providers handle the heavy work. I wanted all aspects of account creation, password reset, etc., to be managed by the providers. All you have to do is log in with your social accounts and start using the site.
